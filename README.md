MTN Nigeria Customer Churn Report – Q1 2025

This project presents a comprehensive visual report on customer churn for MTN Nigeria in Q1 2025 using *Microsoft Excel*. It analyzes churn patterns across dimensions such as:

- Age groups  
- Geographic locations (states)  
- Satisfaction ratings  
- Data usage plans  
- Churn reasons  

Key Insights:
- Osun recorded the highest churn rate.
- Major churn reasons include high call tariffs and better competitor offers.
- Most churned customers are within the 26–45 age range.
- Revenue impact and customer behavior patterns are highlighted visually.

Tools Used:
- Microsoft Excel (for data analysis, charting, icons, and color formatting)
